# Assignment 4 – Intelligent Transport System (Maven Project)

## Overview

This project is a Java Maven application developed for an Intelligent Transport System.  

It includes core domain models (`Driver`, `Bus`), repository classes for file persistence, and both unit and integration tests using **JUnit 5**.

The system demonstrates:

- Object-oriented design
- Data validation rules
- File-based persistence (TXT storage)
- Unit testing
- Integration testing
- Continuous Integration using GitHub Actions
